# AI 任务启动模板

## 1) 任务首行（必填）

```text
启用规则：L1 + L2-模块名
```

示例：

```text
启用规则：L1 + L2-I2C/陀螺仪
```

如果任务不涉及模块细则：

```text
启用规则：L1
```

---

## 2) 任务输入模板（可复制）

```text
任务目标：
涉及模块：
改动边界（只改哪些文件）：
是否允许提交：
是否允许 push：
验证要求：
是否执行精华沉淀判定：
```

---

## 3) 任务输出模板（固定）

```text
问题根因 -> 修改点 -> 验证结果 -> commit id -> 是否 push
通过项：
失败项：
风险项：
下一步动作：
精华沉淀：
沉淀文件：
```
