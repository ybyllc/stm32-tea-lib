## 设置
| 参数      | 值              | 说明                  |
| ------- | -------------- | ------------------- |
| **波特率** | **115200 bps** | 平衡速度与稳定性，WiFi模块默认支持 |
| **数据位** | 8              | 标准配置                |
| **停止位** | 1              | 标准配置                |
| **校验位** | None           | 可加软件校验和             |
| **流控**  | None           | 无线传输不用硬件流控          |
| **帧头**  | 0xAA 0x55      | 双字节帧头，避免误判          |
| **帧尾**  | 0x0D 0x0A      | `\r\n`，便于调试         |

## 数据帧
帧头：0xAA 0x55
帧尾：0x0D 0x0A

| 字节 | 内容       | 说明        |
| ---- | --------- | ----------  |
| 0-1  | 0xAA 0x55 | 帧头         |
| 2    | 0x01      | 命令字：心跳  |
| 3    | 0x04      | 数据长度      |
| 4-7  | uint32 | 数据内容      |
| 8    | checksum  | 校验和 前面数据sum保留1字节 (可选)        |
| 9-10 | 0x0D 0x0A | 帧尾         |

## 控制指令
### 行驶状态
命令字：0x01
内容：1=强制停止！
数据长度：1

示例:
```0xAA 0x55 0x01 0x01 0x01 0x0D 0x0A```

## 传感器数据
### 超声波
命令字：0x11
内容：每路2字节，单位mm，0-65535
数据长度：10

## 摄像头识别通信
### 物体识别的坐标
命令字：0x21
内容：
- CLASS: 物体识别的种类，1-水瓶，2-草，3-其他等等   
- X: 物体的X坐标偏差值，范围-1000~1000.(H和L是高字节和低字节)  
- Y: 物体的Y坐标偏差值，范围-1000~1000.(H和L是高字节和低字节)
- FLAG: 物体识别的状态，0-未识别，1-识别，2-丢失（预测）   
示例：
```AA 55 21 CLASS XH XL YH YL FLAG CHECK 0D 0A```

## 其他
### 喇叭
命令字：0xf1
内容：0=关，1=开
数据长度：1

示例:
```0xAA 0x55 0xf1 0x01 0x01 0x0D 0x0A```
